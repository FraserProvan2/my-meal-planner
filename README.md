## Getting Started
#### Install Composer/Dependencies with (In Order):
1. `composer install`
2. `npm install`

#### Local Server Commands:
- Start: `./vendor/bin/sail up` (http://localhost)
- Stop: `./vendor/bin/sail down`

## Common Development Commands:
#### SCSS/JS Compiling:
- Build: `npm run dev` 
- Watch: `npm run watch`

#### Tests:
- `php artisan test`
